# Anniversaire Enzo

Mini-site statique sans installation ni compilation.

## Déploiement gratuit

- **Netlify** : glissez-déposez ce dossier dans Netlify Drop.
- **GitHub Pages** : déposez ces fichiers à la racine d'un dépôt, puis activez Pages dans les réglages du dépôt.
- **Vercel** : importez le dossier ou le dépôt ; le site est détecté comme site statique.

Pour un aperçu local, ouvrez simplement `index.html` dans un navigateur.
